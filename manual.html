글을 적으면 그냥 글이 적힘.
하지만 어떤 효과를 주고싶다면 테그라는걸 이용해야함.
테그는
<효과명령> 효과를 주고싶은 글자 </효과명령>
형식으로 사용한다.
테그를 쓰고, 글자를 쓰고, 닫는테그를 써야하는데, 닫는테그는 항상 앞에 '/'를 붙여야한다는걸 잊지말자. 안붙였다가 奀된다.

테그명령들
  1. strong테그
    strong테그는 글씨를 굵게할때 사용한다.
    예시 : <strong> 냥파스 </strong>
    결과 : 냥파스
  2. del테그
    strong테그는 취소선을 그을때 사용한다.
    예시 : <del> 냥파스 </del>
    결과 : 냥파스
  3. br테그
    br테그는 줄바꿈할때 쓰는 태그이다. 닫는테그가 필요없다.
    예시 : <br> 냥파스
    결과 :
    냥파스
  4. a테그
    a테그는 링크걸때 쓴다. 특이하게도 href라는 옵션을 넣어서 주소를 지정해준다.
    예시 : <a href="https://m.blog.naver.com/ennkasama/120202992286"> 냥파스 </a>
    결과 : 냥파스
  5. div테그
    div테그는 용도가 무궁무진하지만 DOCL에서는 특수효과를 준다고 생각하면된다. class라는 옵션을 이용하여 효과를 지정해줄 수 있다.
    예시 : <div class="rainbow"> 냥파스 </div>
    결과 :
    냥파스
    사용가능 특수효과들
    rotate, size, rainbow, blink, rainbowrotate, rainbowsize, blinkrainbow, blinkrotate, blinksize, sizerainbowblink
  6. 뭐가 더 많이 있는데, 내가 필요한 거 인터넷에 찾아보면 잘 나온다. 예를 들어 표를 만들고 싶으면 구글에 'html 표 작성 테그'라고 검색하면 table테그가 나올 것이다. 기본적인건 알려줬으니, 필요한건 찾아보도록
